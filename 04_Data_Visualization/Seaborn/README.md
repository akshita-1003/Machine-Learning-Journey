# 🌊 Seaborn

This directory contains my learning journey with **Seaborn**, a statistical data visualization library built on top of Matplotlib.

The notebooks in this folder focus on creating informative and aesthetically appealing visualizations that simplify data exploration and help uncover meaningful insights during Exploratory Data Analysis (EDA).

## Purpose

- Learn Seaborn from fundamentals to advanced concepts.
- Understand statistical data visualization techniques.
- Practice visualizing real-world datasets.
- Develop strong EDA skills for Machine Learning projects.

## Learning Approach

Each notebook generally includes:

- Concept explanation
- Syntax and parameters
- Practical examples
- Visualization interpretation
- Best practices and key takeaways

## Tools & Libraries

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
