# 🌊 Seaborn

This directory contains my learning journey with **Seaborn**, a Python library for statistical data visualization built on top of Matplotlib.

The notebooks in this folder cover various visualization techniques, concepts, and practical examples that are commonly used in Exploratory Data Analysis (EDA) and Machine Learning workflows.

## Purpose

- Learn Seaborn from fundamentals to advanced concepts.
- Practice data visualization using real-world datasets.
- Understand when and how different plots should be used.
- Build a well-documented collection of visualization examples for future revision.

## Learning Methodology

Each notebook in this folder includes:

- Concept explanation
- Syntax and parameters
- Practical examples
- Observations and insights
- Best practices

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
