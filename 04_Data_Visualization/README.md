# 📊 Data Visualization

This folder contains my learning journey in **Data Visualization** using Python. It includes notebooks, examples, and practice exercises to understand how different visualization techniques help in exploring and analyzing data.

## 🎯 Objectives

- Learn the fundamentals of data visualization.
- Understand when to use different types of plots.
- Explore datasets visually before applying Machine Learning models.
- Build intuition for identifying patterns, trends, and outliers.
